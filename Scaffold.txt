│       ├── labels_{train,val,test}.npy    (N,)  0=normal 1=seizure

│   │   — check_credentials()      validates .env before attempting download
│   │   — download_dataset()       wfdb.dl_database("chbmit") → data/raw/
│   │                               returns cached path instantly if already done
│   │   — run_preprocessing()      calls src/data/preprocess.py automatically
│   │   — verify()                 counts patients and processed files, prints status
│   │
│   └── learning_curve.py
│       — Run after training at multiple n_train_patients values
│       — get_best_metrics()       reads best val_f1 and val_recall from log CSV
│       — parse_patient_count()    extracts N from filename e.g. "cnn_lstm_4pat.csv" → 4
│       — main()                   plots F1 and Recall vs patients for all 4 models
│       — Saves: results/learning_curve.png
│
├── src/
│   │
│   ├── data/
│   │   ├── preprocess.py
│   │   │   — parse_summary_file()     parses CHB-MIT annotation .txt files
│   │   │                               returns {edf_filename: [(start_sec, end_sec)]}
│   │   │   — load_edf()               loads .edf via MNE
│   │   │                               returns (num_channels, num_samples) float32
│   │   │                               handles channel count mismatches
│   │   │   — create_windows()         segments continuous EEG into windows
│   │   │                               labels each 1 if seizure overlap ≥ threshold
│   │   │                               stride_sec controls overlap amount
│   │   │   — normalize_windows()      z-score per channel per window
│   │   │   — process_patient()        runs load→window→label for one patient folder
│   │   │   — preprocess()             orchestrates full pipeline:
│   │   │                               processes train/val/test patient groups separately
│   │   │                               computes pos_weight from training patients only
│   │   │                               saves .npy files + meta.json
│   │   │
│   │   └── dataset.py
│   │       — EEGDataset (PyTorch Dataset)
│   │           loads .npy files with mmap_mode="r" (memory-mapped, not all in RAM)
│   │           __getitem__() returns (FloatTensor (23,1280), FloatTensor scalar)
│   │           _augment()    Gaussian noise + channel dropout (training only)
│   │       — get_dataloaders()
│   │           returns train, val, test DataLoader objects
│   │           train: augment=True, shuffle=True
│   │           val/test: augment=False, shuffle=False
│   │           raises clear FileNotFoundError if preprocessing not run yet
│   │       — load_pos_weight()
│   │           loads pos_weight.npy into a torch.Tensor on device
│   │
│   ├── models/
│   │   ├── cnn_baseline.py
│   │   │   — CNNBaseline (nn.Module)
│   │   │       __init__(num_channels, window_samples, out_channels, kernel_size, dropout)
│   │   │       forward(x) → logit (batch, 1)
│   │   │       3× [Conv1d → BatchNorm1d → ReLU → MaxPool1d → Dropout]
│   │   │       → AdaptiveAvgPool1d(1) → Flatten → Linear(64) → ReLU → Linear(1)
│   │   │       Role: ablation baseline — no temporal modeling
│   │   │
│   │   ├── cnn_lstm.py
│   │   │   — CNNLSTM (nn.Module)
│   │   │       __init__(num_channels, window_samples, out_channels, kernel_size,
│   │   │                lstm_hidden, lstm_layers, bidirectional, dropout)
│   │   │       forward(x) → logit (batch, 1)
│   │   │       CNN block (identical to baseline) → permute → LSTM → h_n[-1] → classifier
│   │   │       If bidirectional: cat(h_n[-2], h_n[-1])
│   │   │       Role: primary model — tests whether temporal modeling helps
│   │   │
│   │   ├── cnn_gru.py
│   │   │   — CNNGRU (nn.Module)
│   │   │       __init__(num_channels, window_samples, out_channels, kernel_size,
│   │   │                gru_hidden, gru_layers, bidirectional, dropout)
│   │   │       forward(x) → logit (batch, 1)
│   │   │       Key difference from LSTM: GRU returns (output, h_n) not (output,(h_n,c_n))
│   │   │       Role: GRU vs LSTM ablation — fewer params, similar performance
│   │   │
│   │   └── tcn.py
│   │       — _Chomp1d (nn.Module)
│   │           Removes right-side padding to enforce causal (past-only) convolution
│   │       — _TemporalBlock (nn.Module)
│   │           2× [weight_norm Conv1d → Chomp1d → ReLU → Dropout] + residual connection
│   │           Residual: 1×1 Conv if n_inputs ≠ n_outputs, else identity
│   │       — TCNModel (nn.Module)
│   │           __init__(num_channels, window_samples, num_block_channels, kernel_size, dropout)
│   │           forward(x) → logit (batch, 1)
│   │           Stacks _TemporalBlock with dilation 2^i (1,2,4,8,...)
│   │           Role: parallel alternative to recurrent models
│   │
│   ├── training/
│   │   └── trainer.py
│   │       — Trainer (class)
│   │           __init__(model, train_loader, val_loader, config, model_name, device)
│   │               BCEWithLogitsLoss — pos_weight set in train.py after init
│   │               AdamW optimizer + CosineAnnealingLR scheduler
│   │               GradScaler for AMP (mixed precision)
│   │               Opens log CSV with header: epoch,train_loss,val_loss,
│   │                 val_f1,val_recall,val_roc_auc,lr,grad_norm
│   │           _train_epoch()
│   │               One full pass through train DataLoader
│   │               AMP autocast → loss → backward → unscale → grad clip → step
│   │               Returns (avg_loss, avg_grad_norm)
│   │           _val_epoch()
│   │               One full pass through val DataLoader, no gradients
│   │               Returns (avg_loss, f1, recall, roc_auc)
│   │           train()
│   │               Main loop: _train_epoch → _val_epoch → scheduler → log CSV
│   │               Prints per-epoch table: Loss | F1 | Recall | AUC | LR
│   │               Early stopping on val_f1 (or val_recall per config)
│   │               Saves best checkpoint to checkpoints/{model_name}_best.pt
│   │               Returns best val score
│   │
│   └── evaluation/
│       ├── metrics.py
│       │   — evaluate_model(model, dataloader, device, threshold)
│       │       Runs model on dataloader, returns:
│       │         results dict (accuracy, precision, recall, f1, specificity,
│       │                       roc_auc, pr_auc, threshold, tp/fp/tn/fn, confusion_matrix)
│       │         all_probs (N,) · all_labels (N,) · all_preds (N,)
│       │   — compute_threshold_sweep(all_labels, all_probs)
│       │       P/R/F1 at 50 thresholds 0.05→0.95
│       │       Returns: thresholds, precisions, recalls, f1s, best_f1_threshold
│       │   — print_results_table(results, model_name)
│       │       Prints aligned metric table to console
│       │
│       └── visualize.py
│           — plot_training_curves(log_csv_path, model_name, save_dir)
│               Reads log CSV → plots Loss, F1, Recall vs epoch (train+val)
│               Dashed vertical line at best val_f1 epoch
│               Saves: {model_name}_training_curves.png
│           — plot_grad_norm_history(log_csv_path, model_name, save_dir)
│               Reads grad_norm column from log CSV → log-scale line plot
│               Saves: {model_name}_grad_norms.png
│           — plot_roc_curve(all_labels, all_probs, model_name, save_dir, auc)
│               ROC curve with AUC and random baseline diagonal
│               Saves: {model_name}_roc_curve.png
│           — plot_pr_curve(all_labels, all_probs, model_name, save_dir, pr_auc)
│               Precision-Recall curve with no-skill baseline
│               Saves: {model_name}_pr_curve.png
│           — plot_confusion_matrix(cm, model_name, save_dir, threshold)
│               Seaborn heatmap [[TN,FP],[FN,TP]]
│               Saves: {model_name}_confusion_matrix.png
│           — plot_threshold_sweep(sweep, model_name, save_dir)
│               P/R/F1 across all thresholds, best-F1 marker
│               Saves: {model_name}_threshold_sweep.png
│           — plot_model_comparison(results_dict, save_dir)
│               Grouped bar chart all 4 models × recall/precision/F1/ROC-AUC/PR-AUC
│               Saves: ablation_comparison.png

================================================================
WORKFLOW
================================================================

Step 1 — Setup (run once):
    pip install -r requirements.txt
    # Fill in .env with PhysioNet credentials
    python scripts/setup_data.py

Step 2 — Quick test (edit configs/config.yaml):
    epochs: 2
    n_train_patients: 4
    python train.py

Step 3 — Full training:
    epochs: 50   (early stopping triggers before this)
    n_train_patients: 16
    python train.py

Step 4 — Evaluate:
    python evaluate.py
    # Open results/ folder — check ablation_comparison.png first

Step 5 — Tune threshold if recall is low:
    # Look at results/*_threshold_sweep.png
    # Set evaluation.threshold: 0.3 in configs/config.yaml
    python evaluate.py   # re-run with new threshold

Step 6 — Learning curve:
    # Train at n_train_patients: 4, 8, 12, 16
    # After each run: mv logs/cnn_lstm_log.csv logs/cnn_lstm_4pat.csv
    python scripts/learning_curve.py

================================================================
PYTORCH vs SKLEARN
================================================================

PyTorch                             sklearn
────────────────────────────────    ──────────────────────────────────
All 4 model architectures           recall_score, f1_score
Trainer: forward/backward/AMP       precision_score, accuracy_score
DataLoader + EEGDataset             roc_auc_score
BCEWithLogitsLoss + pos_weight      average_precision_score
AdamW + CosineAnnealingLR           confusion_matrix
GradScaler (mixed precision)
clip_grad_norm_
torch.save / torch.load

Rule: PyTorch for anything on GPU or involving gradients.
      sklearn for metric computation after training is done.
